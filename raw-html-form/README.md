## Description

Simple responsive main HTML form with another linked HTML collecting input values and printing them

## Goals

Practicing on; 

* HTML form elements
* Pseudo classes
* Checking form values(via prebuilt html) at backend

* Used

	* External HTML to store input values

* Not used
